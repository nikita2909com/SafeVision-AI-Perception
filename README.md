
# SafeVision: Autonomous Vehicle Perception with XAI

### Project Overview
SafeVision is a deep learning-based perception system designed for autonomous vehicles. It utilizes **YOLOv8** for real-time object detection and integrates **Explainable AI (XAI)** via Grad-CAM heatmaps to provide visual interpretability of the model's decision-making process.

### Tech Stack
* **Frontend:** React.js (Dark Dashboard)
* **Backend:** Flask (REST API)
* **AI Model:** YOLOv8 Nano
* **XAI:** Grad-CAM (Gradient-weighted Class Activation Mapping)
